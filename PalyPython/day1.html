What is Programming?
Programming is creating a set of instructions for the computer to execute. Here's an example. We can tell the Python logo to draw up, down, left, and right. How do we make it draw a square? Solution: We create a list of instructions to be executed chronologically.
Python Program Example 
number1 = 20
number2 = 10
print("Sum: ", number1 + number2)

# Output
# Sum: 30
Print
In Python, we can display characters in the terminal by using print(). Print is a function so whatever is inside the parentheses will be printed. Characters with quotations are called strings. 
print("Hello Universe!")
            
# Output
# Hello Universe!

Structure
Python uses indentation for blocks in the code, not curly braces like other languages. Tip: Make sure all indents are the same size (Recommended Size : 4 spaces) 
favorite_food = "pizza"
if favorite_food == "pizza":
    print("Your favorite food is pizza")
Variables
Variables are used to store data values. To create a variable, assign a value to the variable name. Tip: You can assign multiple variables in 1 line. 
integer = 4
string = "stringy"
integer, string = 4, "stringy"
Naming Conventions
Use snake_case for variable names in Python.
  • Can only contain alpha-numeric characters
  • Must start with letter or underscore
  • Cannot start with number
  • Case-sensitive
Input
The input function, input(), allows you to receive user input. Write the prompt inside the parentheses. Bonus: Inputted values are strings by default - to receive numbers, convert the input into an integer or float.
name = input("What is your name: ")
print(name, "was here!")
# Output
# What is your name: Nobody
# Nobody was here!
num = int(input("Enter a number: "))
